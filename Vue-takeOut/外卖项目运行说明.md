# 源码项目说明:
	gshop-client_final  客户端应用
	gshop-server_final  服务器端应用


# Vue项目-硅谷外卖的运行说明
## 1. 准备
	1) 确保安装了node环境
	2) 确保安装了mongodb, 并启动了对应的服务

## 2. 启动后台应用
	1). 进入gshop-server_final
	2). 安装依赖包: npm install
	3). 启动应用:npm start

## 3. 启动前台应用并访问
	1). 进入gshop-client_final
	2). 安装依赖包: npm install
	3). 执行命令: npm start
